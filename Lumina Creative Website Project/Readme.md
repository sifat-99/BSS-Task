# Lumina Creative Website Project

**🔗 Live Demo:** [luminacreative-sifat.netlify.app](https://luminacreative-sifat.netlify.app/)

This directory involves a creative agency website project showcasing modern design practices, engaging layouts, and visual storytelling capabilities.

## 📁 Project Structure
- `index.html` - The main homepage for the agency.
- `about.html` - The about section showcasing the team.
- `contact.html` - Interactive contact section.
- `style.css` - Custom styling and CSS layout rules.
- `images/` - Visual graphics and project assets.

## ✨ Features
- Modern, clean aesthetic tailored towards creative agencies.
- Smooth navigation between multiple internal pages.

## 🚀 Getting Started
Open `index.html` in your browser to view the landing page locally and navigate through the content.
