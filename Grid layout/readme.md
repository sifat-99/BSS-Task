# Grid Layout

An advanced design module detailing challenges and exercises for mastering 2-Dimensional interface architectures using CSS Grid.

## 📁 Tasks & Contents
This directory contains focused exercises breaking down complex dimensional bounds:
- **Practice:** Using fractional (fr) spacing rules, precise column/row span logic, template areas, and placing items specifically on grid lines.
- **Challenge:** Constructing a massive, abstract, multi-column dashboard strictly relying on native CSS Grid layout directives.

## 🚀 Getting Started
Simply open the HTML files inside the `practice` or `Challange` folders in your browser and open layout devtools to visualize the grid geometry.
