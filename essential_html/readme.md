# Essential HTML

A practical module detailing basic exercises to understand how structural HTML is put together. 

## 📁 Tasks & Contents
This directory teaches foundational HTML principles:
- **Syntax and Structure:** How standard tags are written.
- **Skeleton of a web page:** Working with standard boilerplate elements such as head, body, and titles.

## 🚀 Getting Started
Explore the contained HTML files and open them in any standard web browser to review the visual structure versus the code block.
