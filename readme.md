## 📚 Learning Progression & Modules

A comprehensive step-by-step series of practical exercises covering all aspects of frontend development. These modules serve as standard building blocks for mastering web capabilities.

### Step 1: HTML Fundamentals
- [**Essential HTML**](./essential_html): Basic structure, syntax, and skeleton of web pages
- [**More HTML Elements**](./More%20HTML%20Elements): Deep dive into semantic tags, metadata, and structural components
- [**HTML Form Input Elements**](./Html%20form%20input%20elements): Comprehensive exercises to master interactive forms, form validation, and complex user inputs

### Step 2: CSS & Styling Core
- [**CSS Basics**](./CSS%20Basics): Properties, selectors, cascading rules, colors, and foundational styling
- [**Box Model**](./Box%20Model): Mastering the conceptual box that wraps HTML elements – understanding margins, borders, padding, and content dimensions
- [**Advanced Selectors**](./Advanced%20Selectors%20Pseudo%20Classes%20Elements): Exploring high-level targeting utilizing pseudo-classes (e.g., `:hover`, `:nth-child`) and pseudo-elements (e.g., `::before`, `::after`)
- [**Various CSS Features**](./Various%20CSS%20Features): Filters, blend modes, shadows, and utilizing modern aesthetic web styling properties

### Step 3: Advanced Layouts & Responsive Design
- [**Flexbox**](./Flexbox): Practical exercises showcasing 1-dimensional layouts, alignment, flex behaviors, and spacing distributions
- [**Grid Layout**](./Grid%20layout): Structuring advanced 2-dimensional layouts and container systems for complex user interfaces
- [**Responsive Design**](./Responsive): Implementing media queries, fluid units, and responsive layouts to ensure cross-device consistency

### Step 4: Animations & Interactivity
- [**Transitions, Animation & A Little JavaScript**](./Transitions%20Animation%20A%20Little%20JavaScript): Bringing standard, static web pages to life. This module covers implementing keyframes, smooth CSS transitions, interactive 3D transforms, and leveraging basic JavaScript constraints to trigger effects

### Step 5: Web Standards & Accessibility
- [**Web Accessibility Introduction**](./Web%20Accessibility%20Introduction): Understanding the necessity of ensuring web content is accessible to all users. Focuses on ARIA labels, semantic structure benefits, standard color contrasts, and keyboard navigability
