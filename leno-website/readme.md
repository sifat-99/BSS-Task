# Leno Website Project

This directory contains the source code and assets for the **Leno App** project. It is a fully functional, responsive landing page for showcasing mobile applications.

## 📁 Project Structure
- `index.html` - Main landing page.
- `details.html` - Supplemental page with more details.
- `css/` - Contains the styling and layouts.
- `images/` - Visual assets used across the web page.
- `script.js` - Interactive functionality across the site.

## ✨ Features
- Fully responsive layout
- Modern styling and structural elements
- Clean, maintainable file structure

## 🚀 Getting Started
Open `index.html` in your browser to view the project locally, or access the live deployment linked in the main repository README.
