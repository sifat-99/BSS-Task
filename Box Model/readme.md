# Box Model

A dedicated block of exercises detailing challenges to master parsing structural geometry via the CSS Box Model.

## 📁 Tasks & Contents
This directory contains focused exercises breaking down borders, padding, and margins:
- **Practice:** Sandbox files where you adjust elements to understand how margin collapsing works, standard dimensional tracking, and `box-sizing` properties.
- **Challenge:** Creating cohesive blocks where spacing accurately mirrors a target design.

## 🚀 Getting Started
Simply open the HTML/CSS pairings within the browser to inspect element geometry.
