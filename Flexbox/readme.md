# Flexbox

A comprehensive module detailing challenges and exercises for mastering 1-Dimensional Flexible Box layouts.

## 📁 Tasks & Contents
This directory heavily utilizes the `display: flex` architecture:
- **Practice:** Understanding the main/cross axes, alignment rules, justified layouts, and item space distribution.
- **Challenge:** Building common interface patterns, like responsive navbars, distinct component alignment, and card spacing arrays using only Flexbox logic.

## 🚀 Getting Started
Simply open the HTML files inside the `practice` or `challange` folders to view the structures bending organically to constraints.
