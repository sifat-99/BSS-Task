# Transitions, Animation & A Little JavaScript

**🔗 Presentation Website Live Demo:** [presentation-website-pink.vercel.app/#page-2](https://presentation-website-pink.vercel.app/#page-2)
**🔗 Hamburger Menu Live Demo:** [bss-task-zvty.vercel.app](https://bss-task-zvty.vercel.app/)

A creative interactive module detailing exercises on bringing standard DOM nodes actively to life.

## 📁 Tasks & Contents
This directory jumps away from standard static presentations:
- **Practice:** Working with CSS properties such as smooth timing functions (`transition`), distinct multi-stage keyframe loops (`@keyframes`), and intricate depth rendering (`3d transform`). 
- **Challenge:** Adding standard DOM selectors using slight vanilla JavaScript scripts to trigger animation clauses, open drawers, and apply/remove specific reactive classes when nodes are queried.

## 🚀 Getting Started
Simply open the respective project folders in any browser to watch the interactive transformations execute perfectly.
