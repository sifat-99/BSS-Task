# CSS Basics

**🔗 Challenge Live Demo:** [bss-task-1nid.vercel.app](https://bss-task-1nid.vercel.app/)

A robust module detailing the foundational challenges and exercises for mastering core Cascading Style Sheets rules. 

## 📁 Tasks & Contents
This directory focuses on the start of the design journey:
- **Practice:** Selectors syntax, specific styling definitions, manipulating text, and adding color and contrast.
- **Challenge:** Problem statements requiring you to style a previously plain document entirely with an external CSS file.

## 🚀 Getting Started
Open the HTML files in a browser to review the CSS in action.
