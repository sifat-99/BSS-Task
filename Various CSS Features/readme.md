# Various CSS Features

A module detailing modern, robust styling utilities outside basic layouts and color blocks.

## 📁 Tasks & Contents
This directory explores visual aesthetic capabilities available natively in CSS without external image processing:
- **Practice:** Box shadows, drop shadows, blend modes, variable definitions, and standard filtering overlays.
- **Challenge:** Build an interactive component utilizing several filter changes on state adjustments.

## 🚀 Getting Started
Open the files to see these CSS aesthetic tricks in action dynamically.
